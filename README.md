## Fitness


### 一、为什么建立这个计划


身体不好是废品，代码写的再好，你也得要有个强健的体魄。难道你想30岁就把身体搞垮？考虑程序员这个群体的特殊性，希望与广大程序员共勉，找到一个循序渐进的锻炼方法，立此计划。

### 二、程序员狗的一生


>程序狗的一生：22岁进外包公司，每天coding到3点，近视了；25岁进IT公司任高级工程师，天天坐着成了胖子，患前列腺炎；28岁进互联网公司担任架构师，贷款买房，同年秃顶；33岁成为CTO，熬出一身病，同年还完房贷；35岁带领创业公司上市前期发现癌症晚期，挂了，同年，老婆带着孩子房子改嫁。

看完以上你有啥感想？

改变现状，工作再忙也要抽时间健身，身体健康才是最重要的！
